Gnome3 显示/隐藏 桌面图标的小脚本

Apr 15, 2015 • admin

categories : 杂货

扫描二维码

有时候会嫌弃桌面上的图标影响了美观,就想隐藏图标,好好欣赏桌面背景….但是去tweak设置太过麻烦.

去dconf-editor 查看show-desktop-icons 这一项的scema的名字,使用个gsettings set 来设置该项的值

脚本参考 https://gist.github.com/darthvish/07b9aca21d598a89486a 

#!/bin/bash
output=$(gsettings get org.gnome.desktop.background show-desktop-icons)
echo "show-desktop-icons is $output"
if [ "$output" == "true" ]; then
    gsettings set org.gnome.desktop.background show-desktop-icons false
    echo "hide"
else
    gsettings set org.gnome.desktop.background show-desktop-icons true
    echo "show"
fi

将脚本拷贝到/usr/bin/ 或者其他目录下,给予可执行权限.再在系统设置中给它添加一个快捷键(如 Window + D),就可以愉快的使用了
