记录下折腾和学习的过程

2017新年杂谈

Jan 2, 2017 • admin

categories : 杂谈

扫描二维码

每年都会写一篇新年杂谈，过去一年格外懒，一共没写几篇博客。 趁元旦假期的最后一天，抓紧时间开写，免得打破这个习惯。

过去一年多的时间里一直为找一份糊口的工作四处奔波，没有想到最后又回到了北京。 第一次在北方的过冬，在暖气基本等于没开的小黑屋里，看着窗外雾霾里若隐若现的路灯，和马路上冒着热气的井盖。 感觉就像看到福特告诉阿瑟地球马上就要被沃贡人毁灭了一样荒诞。

昨天在路上，同事说，跟我说话时，我总是三句不离计算机。不是在说工作，就是在说游戏。 听到他说才突然发现，我已经彻底变成一个无趣的人了。很多时候都在盼望，我的笔记本能和我聊聊天。 当然，希望它是小叽而不是Mike。

之前看了一部风格很独特的动画，叫《四叠半神话大系》，里面有这样一句话， “我们的大部分苦恼，都是始于梦想着得到别样的人生，寄希望于自己可能做到的事情，这正是万恶的根源。除了现在的自己，你不能成为其他的任何人。”

是啊，除了现在的自己，你不能成为其他的任何人。 能做的不过是做好现在的工作，过好现在的生活。

要说2016有什么值得纪念的话，就是在六月底毕业了，结束了多年的学生生涯。 在大学四年里，我一直都在尽力享受混吃等死的生活，毕竟这种生活十分难得。 白天在湖边晒太阳喂喂鱼。晚上在阳台吹着风数星星，无聊了半夜拿两本上楼读，没事在走廊喊两嗓子就能叫上一大堆人下楼打球。 连写程序也可以想到什么就做什么，完全不考虑投入产出，看书也能挑厚的一页页慢慢看。

四年时间，我终于不再是那个沉迷Dota的电竞少年，成功变成了一个死宅。真是一个悲伤的故事。

之前第一份正式工作去了豹厂，本来是想告别安全行业，从做广告开始，开启一个正常的开发生涯的。 然而世事难料，没过两个月就遇到了项目组解散。

不过来了数字后，感觉也还不错，同事们很好相处，项目也比较有意思，接触了很多以前没搞过的东西。 快糙猛的东西做多了之后，停下来做细致一点的，追求单机高性能的项目也挺有意思的，顺便也重新学习了下C++ (其实主要还是C，最多算是C with Class, 专門写底层的人写的C++代码和写应用的人写的感觉完全不是一回事。感觉就是C写烦了换种语言玩玩而已)。 不过还是写Python最开心，毕竟人生苦短嘛。 和之前不同的是，平时有很大一部分时间都在和别的岗位的同事交流，感觉沟(shi)通(bi)技能提升了很多。

去年年初列的计划里，准备看的书倒是基本都看完了，bitbuck 上的几个小项目也有一些进展，分布式相关的东西也看了一大堆， 不过偏底层的编译和系统相关的东西进展有点缓慢。今年准备多搞搞这两方面。还有就是继续读几个开源项目的源码。

最后祝大家新的一年都能过的开心，CN Dota TI7 再创辉煌 ： ）
